# Version 34 — Duplex PDF and Expanded Pitching Layout

Version 34 was built from the approved Version 33 production source.

## Approved layout

1. Page 1 retains the complete scorecard and scoring grids.
2. Game Notes move to page 2 for two-sided printing.
3. The former Game Notes area is removed from page 1.
4. Away and Home pitching tables expand across the full page width.
5. Six pitcher rows remain available for each team.
6. Pitching columns are limited to Pitcher / No., IP, H, R, ER, BB, K, HBP, and WP.
7. The top-right Replay/ABS area now shows readable availability rather than unexplained token boxes.
8. Page 2 combines Game Notes and Game Event Notes into one large “Game Notes & Events” section, while retaining review totals and the separate Review Decision Log.
9. Extra-inning scorecards are followed by a continuation notes page so duplex pairing remains intact.
10. All Version 33 scoring and data-persistence behavior is preserved.
